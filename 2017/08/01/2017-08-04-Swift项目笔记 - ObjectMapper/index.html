关于处理请求来的数据模型, 之前写OC的时候, 一直在使用JSONModel, 后来朋友推荐YYKit下的YYModel, 它的效率和上手的效果着实让我神清气爽, 而在Swift的项目中, 我则推荐使用ObjectMapper. 下面的文字是对ObjectMapper的一些整理和记录, 方便日后回顾.

ObjectMapper

ObjectMapper is a framework written in Swift that makes it easy for you to convert your model objects (classes and structs) to and from JSON.

正如ObjectMapper上说的一样, 是一个用Swift编写的一个处理JSON与Model object转换的库.

  • Model to JSON
  • JSON to Model
  • 嵌套(in Array or in Dictionary)
