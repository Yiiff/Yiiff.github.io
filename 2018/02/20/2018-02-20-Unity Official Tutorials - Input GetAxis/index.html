返回值类型: Float

Axis影响因素

  1. Gravity(重力)
    • 其值影响return to 0的速度，Gravity越大，恢复速度越快;
  2. Sensitivity(灵敏度)
    • 与Gravity相反，其值影响response to 1/-1的速度，Sensitivity越大，响应速度越快;
  3. Dead(死区)
    • Size of the analog dead zone. All analog device values within this range map to netural.
    • 当我们使用操纵杆🕹️（Joystick）来表示我们的Axis时，我们并不想因为摇杆很小范围的移动来影响我们的Axis，为了避免这种情况，我们可以设定Dead，其值越大，Dead Zone 越大;
  4. Snap
    • Allow to return to 0, if both positive button and negative button was held;
